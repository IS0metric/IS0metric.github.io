Python Virtual Environments - With PowerShell Helper Script and Desktop Shortcuts

Navigate to envs folder, Project root, then call the venv instruction:

python -m venv [name]

Navigate to project dir and generate the PowerShell script named env.ps1:

& [path to env]\Scripts\activate.ps1

Create the Atom shortcut and adjust the path:

[user path]\AppData\Local\atom\atom.exe "[path to project]"

Create the PowerShell shortcut and adjust the path with the script execution command:

C:\Windows\System32\WindowsPowerShell\v1.0\powershell.exe -noexit -command "cd [path to project]; .\env.ps1"
